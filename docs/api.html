Emoji version¬∂

Every emoji in emoji.EMOJI_DATA has a version number. The number refers to the release of that emoji in the Unicode Standard. It is stored in the key 'E'. For example the emoji ü•á :1st_place_medal: is version E3.0 that is Emoji 3.0 or Unicode 9.0:

>>> emoji.EMOJI_DATA['ü•á']['E']
3

For more information see http://www.unicode.org/reports/tr51/#Versioning
